# Changelog

All notable changes to this project are recorded here.

## 0.1.0 - 2026-05-15

- Established the Astro static site baseline for Chong Lab.
- Added the Cloudflare Pages deployment workflow.
- Added the design system document and maintenance guide.
- Added the first editorial static pages for the lab.

## 0.2.0 - 2026-05-15

- Aligned the public site with the Chong Lab agent specification.
- Reworked the homepage, research, people, and datasets pages around the editorial static layout.
- Added public headers, redirects, navigation scripts, and maintenance notes for future sessions.

## 0.2.1 - 2026-05-15

- Added system-aware light/dark theming with a user toggle.
- Rebalanced the site palette so it can read clearly in both themes.

## 0.3.0 - 2026-05-15

- Added locale-prefixed pages for Japanese and Chinese alongside the English default.
- Introduced shared page rendering and locale-aware navigation, SEO, and footer links.
